Open Source Code

I am a strong supporter of the open source idea and make virtually all of my research and utility code available. In particular, I try to pay attention to produce modular and re-usable to code to enable the community to extend and build upon the existing frameworks. So far, my software has received over 2k stars and 300 forks on github. Please be invited to use, re-use, modify, and contribute to any of the projects!
